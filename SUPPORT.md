# Support

Use the right channel so issues stay actionable:

* **Setup help and usage questions:** [GitHub Discussions](https://github.com/puukis/streamd/discussions)
* **Reproducible bugs:** [GitHub Issues](https://github.com/puukis/streamd/issues/new/choose)
* **Security reports:** follow [SECURITY.md](SECURITY.md)

When asking for help, include:

* host OS, compositor, GPU, and driver version
* client macOS version and hardware
* whether the problem is LAN-only or also happens over WAN
* relevant `RUST_LOG=debug` output from both sides
